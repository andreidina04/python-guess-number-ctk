A simple Python game with a graphical interface using **CustomTkinter**.  
The goal is to guess a randomly generated number between 0 and 101, depending on the chosen difficulty level.

---

## Features

- Choose difficulty: Easy (10 attempts), Medium (5 attempts), Hard (3 attempts)
- Score system that increases/decreases based on guesses and hint usage
- Visual feedback: number is higher/lower, win or lose messages
- Hint: shows a range where the number is located

---

## Requirements

- Python 3.8+  
- [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter)

Install dependencies with:

```bash
pip install customtkinter

git clone https://github.com/username/guess-the-number.git
